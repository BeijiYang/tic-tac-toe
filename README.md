### 根据 [React官方教程](https://facebook.github.io/react/tutorial/tutorial.html) 完成的井字棋（tic-tac-toe）小游戏。
#### 这是 [我翻译的教程](https://beijiyang.github.io/react-tutorial-translation/)gitbook


实现功能：
* 可以玩井字棋游戏；
* 当有玩家获胜时，宣布结果；
* 存储棋局的历史步骤记录；
* 允许玩家穿越回之前，查看当时棋盘的格局；

扩展添加的功能
* 增加和棋判断；
* 记录落子的位置时，以“（1，3）”的格式显示，而不仅仅只显示“6”；
* 高亮显示胜负手，使结局一目了然。
